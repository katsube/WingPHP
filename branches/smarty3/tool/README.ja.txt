1. makedoc.bat - ドキュメント生成用バッチ
￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣
  ○これは何？
    ・phpdocを実行するバッチファイルです。
	・実行すると [WingPHPルート]/doc 内にドキュメントを出力します。
  
  ○インストール
    ・動作には phpDocumentor が必要です。
    ・phpDocumentorのインストール方法は以下です。
       $ pear install phpdocumentor
	・コマンドラインから phpdoc コマンドを叩きUsageが表示されるか確認します。
	  もしされない場合はPATHが通っているか確認を。
	・Windows環境ではこれ以上特にすることはありません。
	・UNIXライクなOSでは、拡張子を変更、実行権限を付与、1行目に #!/bin/sh
	  を追加します。

  ○実行
    ・Windows環境ではダブルクリック、もしくはコマンドプロンプトから
	  ファイル名を指定して実行します。
	・UNIXライクなOSでも同様です。

  ○その他
	・phpDocumentor公式
      http://www.phpdoc.org/

2. conv - 文字列変換
￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣
 ※このツールはベータ版です。
   将来的に仕様が大幅に変更になる可能性があります。


  ○これは何？
    ・コマンドラインから与えた文字列を任意の形式に変換します。

  ○インストール
    ・動作には コマンドライン版のPHP(5.3以上) が必要です。
  
  ○実行
    ・コマンドラインから以下のように実行します。
	  $ php conv {実行する処理名} {文字列}

	・実行する処理名には以下のいずれかの文字列を指定します。
      sha1
	  md5
	  base64_encode
	  base64_decode
	  urlencode
	  urldecode
	  html

	・実行例
	  $ php conv html "<b>"
      &lt;b&gt;
	  
	  $ php conv sha1 "apple"
      d0be2dc421be4fcd0172e5afceea3970e2f3d940

3. sqldo - SQL実行
￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣
 ※このツールはベータ版です。
   将来的に仕様が大幅に変更になる可能性があります。

  ○これは何？
    ・WingPHPの設定に従ってSQLを実行します。

  ○インストール
    ・動作には コマンドライン版のPHP(5.3以上) が必要です。

  ○実行
    ・SQLを記録したファイルを所定の場所にセットします。
	    ※wingphp/tool/sql/do 配下に設置します。
		※テスト用のサンプルファイルがあるので参考に
		  してください。
	
	・SQLファイルは次のようなネーミングルールに従います。
	
	    [グループ識別子]_ファイル名.sql


    ・コマンドラインから以下のように実行します。
      $ php sqldo [グループ識別子]

	・実行例
      $ php sqldo teststart
      ----------------------------
      *** SQL do
      ----------------------------
        Group:teststart
      Account:master
      ----------------------------
      -- ./sql/do/teststart_createtable.sql
      -- ./sql/do/teststart_insert.sql

	  $php sqldo testend
      ----------------------------
      *** SQL do
      ----------------------------
        Group:testend
      Account:master
      ----------------------------
      -- ./sql/do/testend_droptable.sql
