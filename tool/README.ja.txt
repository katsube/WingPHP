1. makedoc.bat - ドキュメント生成用バッチ
￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣
  ○これは何？
    ・phpdocを実行するバッチファイルです。
	・実行すると [WingPHPルート]/doc 内にドキュメントを出力します。
  
  ○インストール
    ・動作には phpDocumentor が必要です。
    ・phpDocumentorのインストール方法は以下です。
       $ pear install phpdocumentor
	・コマンドラインから phpdoc コマンドを叩きUsageが表示されるか確認します。
	  もしされない場合はPATHが通っているか確認を。
	・Windows環境ではこれ以上特にすることはありません。
	・UNIXライクなOSでは、拡張子を変更、実行権限を付与、1行目に #!/bin/sh
	  を追加します。

  ○実行
    ・Windows環境ではダブルクリック、もしくはコマンドプロンプトから
	  ファイル名を指定して実行します。
	・UNIXライクなOSでも同様です。

  ○その他
	・phpDocumentor公式
      http://www.phpdoc.org/
