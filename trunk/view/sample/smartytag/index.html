{markdown} Smarty用 カスタムプラグイン --------------------------- WingPHPでは独自のカスタムプラグインをいくつかご用意しています。 ## ### request 指定した$_REQUESTの値を表示します。 ```smarty {literal}{request key="foo" escape="html|url"}{/literal} ```
key属性(必須)
key="foo" とすると、$_REQUEST["foo"]と同義になります。
escape属性(任意)
未指定の場合か、"HTML"を指定した場合は htmlspecialchars()、 "url"を指定した場合は urlencode()
### markdown markdown記法に変換します。 ```smarty {literal}{markdown} 見出し ======= ここに本文を書いたりします。 * ぬこ * ねこ * キャット {/markdown}{/literal} ``` ※オプションはありません。
機能 文法 実行結果 備考
リスト * ねこ
* ぬこ
* キャット
  • ねこ
  • ぬこ
  • キャット
 *の代わりに+, -でも同様の結果になります。
また空行を挟むことで、リスト中でも改行することができます。
リスト (順番) 1. ねこ
2. ぬこ
3. キャット
  1. ねこ
  2. ぬこ
  3. キャット
 数値を記述すればよいので、順番が入れ替わっても、すべて同じ数字でも動作します。
リンク [Google](http://www.google.com/)
[Google](http://www.google.com/)"これはGoogle" 		Google Google(title付き) 以下のように後方で定義することもできます。
代表的な検索エンジンには [Google][1] や [Yahoo][2]、[bing][3] があります。

[1]: http://www.google.com/   "Google.com"
[2]: http://www.yahoo.co.jp/  "Yahoo!Japan"
[3]: http://www.bing.com/     "bing"

代表的な検索エンジンには Google や Yahoo、bing があります。
